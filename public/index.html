    httpstatus.it

    a free status code & responses generator

    Welcome!

    I built this tool to test a couple of things when working in my development environment, in order to test some external service responses. I thought it could be useful to have a status code and responses generator always available without worrying about anything.

    Feel free to use it... for free! :)

     

    How to Use It:

    • type your desired status code after the base URL of this site;
    • use that URL;
    • ta-dah!

    Note: if you want, you can also add one or more segments after the status code. Something like http://httpstatus.it/400/my/custom/segment will be perfectly fine.
